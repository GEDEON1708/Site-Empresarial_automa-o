# Autoflow

Site institucional da Autoflow, desenvolvido para apresentar servicos de automacao empresarial, integracao de sistemas e melhoria de eficiencia operacional com uma comunicacao mais profissional.

## Autor

Gedeon

## Rodar localmente

**Pre-requisito:** Node.js

1. Instale as dependencias:
   `npm install`
2. Rode o servidor:
   `npm run dev`
3. Abra:
   `http://localhost:3000`
